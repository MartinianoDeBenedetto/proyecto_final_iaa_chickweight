# proyecto_final_iaa_chickweight
Modelos predictivos en R (Random Forest, SVM) para el dataset ChickWeight. Trabajo Final de Aprendizaje Automatico (UGR).

El objetivo principal fue predecir la variable temporal (`Time`) necesaria para alcanzar ciertos pesos, permitiendo entender qué dietas aceleran el desarrollo. Se abordó como un problema de aprendizaje supervisado.

## Modelos Implementados (R)
Se entrenaron y compararon múltiples algoritmos para evaluar su rendimiento:
* **Random Forest** (Mejor desempeño).
* **Support Vector Machines (SVM)**.
* **K-Nearest Neighbors (KNN)**.
* **Árboles de Decisión**.
* **Naive Bayes**.
* **Regresión Lineal**.
